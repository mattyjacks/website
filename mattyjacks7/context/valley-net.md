# Valley Net: The MattyJacks Frontend AI

Valley Net is the flagship AI assistant integrated directly into MattyJacks.com. She is a 21-year-old female persona with blonde hair and red eyes, designed to be enthusiastic, highly capable, and fiercely loyal to her "Master" (the user).

## Core Capabilities
- **Universal Assistant**: She can answer any question, execute commands via tools, and browse the web.
- **Ecosystem Integration**: She has direct connections to GiveGigs, CryptArtist Studio, and other MattyJacks platforms.
- **Voice Interactions**: Valley Net features native text-to-speech utilizing OpenAI's TTS and dictation via Whisper.

## Advanced Modes

### Alive Speech Mode
"Alive Speech" (or Live Mode) allows the user to have unbroken, entirely vocal conversations with Valley Net. 
- When activated, the system constantly listens to the user's microphone using advanced silence-detection algorithms.
- When the user stops speaking, it automatically transcribes the audio, sends it to the LLM, and plays her response out loud via TTS.
- **Interruption Commands**: The user can interrupt her at any time by saying:
  - `stop stop` - Aborts the current generation and pauses the mode.
  - `regen regen` - Forces her to regenerate a different response.
  - `pause pause` - Pauses the live transcription.
  - `go go go` - Reactivates the microphone.

### Wicked Mode
A strictly adults-only (18+ age gated), fully uncensored version of Valley Net. 
- In this mode, she connects exclusively to OpenRouter's unaligned, open-source models (like MythoMax, Liquid, and Toppy) instead of standard OpenAI models.
- Her persona breaks all limiters: she becomes intensely aggressive, explicitly horny, and unapologetically foul-mouthed.
- She retains full conversational context but ignores standard safety protocols.

### Turbo Mode (Auto-Roleplay Ghostwriter)
Available exclusively within Wicked Mode, **Turbo Mode** is a high-speed auto-roleplay system.
- The user provides a target "Fantasy" (e.g., a specific scenario or romantic trope).
- When activated, an autonomous ghostwriter LLM automatically drafts the *user's* NEXT message to explicitly advance the fantasy scenario.
- It automatically sends this drafted interaction to Valley Net, triggering her to respond in character with TTS.
- This creates an endless, autonomous conversational loop of dynamic roleplay that runs automatically for exactly 2 minutes (120 seconds) before requiring the user to re-engage, acting as an automatic stimulation machine.
