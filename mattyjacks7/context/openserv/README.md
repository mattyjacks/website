# OpenServ Documentation & Context Library

This folder contains comprehensive documentation for OpenServ.ai - a platform for building autonomous AI agents with advanced cognitive capabilities.

## Structure

- `sdk/` - OpenServ TypeScript SDK documentation and API reference
- `agent-tutorial/` - Agent development tutorials and examples
- `agent-examples/` - Real-world agent implementations
- `api-docs/` - API endpoint specifications
- `concepts/` - Core concepts and architecture
- `examples/` - Code examples and patterns
